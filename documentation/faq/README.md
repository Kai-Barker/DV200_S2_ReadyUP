# Frequently Asked Questions

### Q: How do I create an LFG post?

- A: You must be logged in to create a post. First, navigate to the game category you want to post in, then click the "Create Post" button. This will open a modal where you can fill in your group's details, including a title, description, start time, and relevant tags.

### Q: : Why can't I find the game I want to play?

- A: All game categories are added by the site administrators. If you don't see your favorite game, it just means it hasn't been added to the database yet.

### Q: How do I delete a post I created?

- A: As the site is still in beta this feature is yet to be added. You may email support to have the post deleted [Support](mailto\:241065@virtualwindow.co.za)

### Q: How do I report an offensive/spam post?

- A: As the site is still in beta this feature is yet to be added. You may email support to have the post deleted [Support](mailto\:241065@virtualwindow.co.za)

### Q: What is the CheatCode? Can i sign up without one?

- A: Think of the CheatCode like a secondary password thats been implemented as a creative multi factor authentication method. The application requires this CheatCode on all users.
